# Playwright TypeScript Boilerplate 🚀

Este es mi template profesional para proyectos de QA Automation.

## 🛠️ Estructura del Proyecto
- `tests/`: Scripts de prueba.
- `page-objects/`: Clases bajo el patrón Page Object Model.
- `.github/workflows/`: Configuración de CI/CD con GitHub Actions.

## 🚀 Cómo empezar
1. Haz clic en **"Use this template"**.
2. Clona tu nuevo repo.
3. Instala dependencias: `npm install`.
4. Instala navegadores: `npx playwright install`.

## 📈 Comandos útiles
- Ejecutar tests: `npx playwright test`
- Ver reporte: `npx playwright show-report`
- Ejecutar en modo UI: `npx playwright test --ui`
